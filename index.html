What is an Unreal Engine?

Unreal Engine is a fairly popular and in-demand game engine developed by Epic Games and released in 2014. It has been used to develop world famous and popular games such as Borderlands 2, Gears of War, Bioshock Infinite. This tool allows developers to easily create high-quality graphics using advanced features such as real-time lighting and physically based shading to create visuals in a realistic format.
